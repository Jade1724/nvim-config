# My Neovim config 

## Setup

Copy `init.vim` under `.config/nvim/`.

Install neovim. `brew install neovim` for Mac.

Get vim-plug from https://github.com/junegunn/vim-plug. 

`nvim {folder}` to launch neovim. There will be missing plugin errors when first time launching.

Run `:PlugInstall` to install plugin.

Quit and reopen neovim.
